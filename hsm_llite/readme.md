# MSEC MCU Secure Drivers - HSM Lite Driver (CAM)

The HSM Lite CAM Driver provides access to the Crypto Acceleration Module available in PIC32 devices with HSM Lite hardware.

The functionalities provided by the HSM Lite Driver are the following:
- AES encryption/decryption
- ECDSA sign/verify
- ECDH key agreement
- Hashing (SHA-2 family)
- Public Key Engine (PKE) operations
- True Random Number Generation (TRNG)
